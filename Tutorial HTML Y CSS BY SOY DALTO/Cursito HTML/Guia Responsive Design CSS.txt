- BLOQUES FLEXIBLES


! Para que el contenido tenga la resolucion que queremos podemos utilizar el max-width o min-width !

/max-width: darle un maximo para que se estire/

/min-width: se encoje hasta un minimo/

! El problema de definir un height fijo es que el contenido se desvorda, es mejor utiliza la propiedad min-height para que el contenido se adapte.!

! Si queremos que los elementos se adapten usamos la papi flex box !
